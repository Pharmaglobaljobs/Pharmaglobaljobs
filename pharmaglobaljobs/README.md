# PharmaGlobalJobs 💊

International pharmaceutical job portal for Indian B.Pharm freshers.

## Deploy to Vercel (Free - 3 steps)

1. Upload this folder to GitHub (new repo)
2. Go to vercel.com → New Project → Import repo
3. Click Deploy — live in 2 minutes ✓

## Run Locally

```bash
npm install
npm start
# Opens at http://localhost:3000
```

## Features
- 🔍 Job search with filters (country, visa, fresher-only)
- 📝 Apply flow with AI cover letter generator
- 📄 Resume builder with live preview
- 🎤 Interview prep (10 questions + AI mock interviewer)
- 🤖 AI Career Advisor (powered by Claude)
- 🚀 Deploy guide built-in
